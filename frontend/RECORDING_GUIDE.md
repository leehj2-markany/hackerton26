# 데모 시나리오 녹화 가이드

## 준비 사항

### 1. 화면 녹화 도구 설치

#### macOS 기본 도구 (추천)
- **Cmd + Shift + 5** - 화면 녹화 시작
- 별도 설치 불필요

#### GIF 변환 도구
```bash
# Homebrew로 ffmpeg 설치
brew install ffmpeg

# 또는 gifski 설치 (고품질 GIF)
brew install gifski
```

### 2. 브라우저 준비
- Chrome 또는 Safari 실행
- http://localhost:3000/ 접속
- 개발자 도구 닫기 (깔끔한 화면)
- 브라우저 창 크기 조절 (1280x720 권장)

## 녹화 후 처리

### 1. 동영상 → GIF 변환 (ffmpeg)

```bash
# 기본 변환 (파일 크기 큼)
ffmpeg -i input.mov -vf "fps=10,scale=1280:-1:flags=lanczos" output.gif

# 최적화 변환 (파일 크기 작음, 품질 좋음)
ffmpeg -i input.mov -vf "fps=15,scale=1280:-1:flags=lanczos,split[s0][s1];[s0]palettegen[p];[s1][p]paletteuse" output.gif

# 더 작은 파일 (해상도 낮춤)
ffmpeg -i input.mov -vf "fps=10,scale=800:-1:flags=lanczos,split[s0][s1];[s0]palettegen[p];[s1][p]paletteuse" output.gif
```

### 2. 동영상 → GIF 변환 (gifski, 고품질)

```bash
# 1. 동영상을 프레임으로 추출
ffmpeg -i input.mov -r 15 frame%04d.png

# 2. gifski로 GIF 생성
gifski -o output.gif frame*.png --fps 15 --quality 90

# 3. 프레임 삭제
rm frame*.png
```

### 3. GIF 최적화 (파일 크기 줄이기)

```bash
# gifsicle 설치
brew install gifsicle

# GIF 최적화
gifsicle -O3 --colors 256 input.gif -o output_optimized.gif

# 더 작게 (색상 줄이기)
gifsicle -O3 --colors 128 input.gif -o output_small.gif
```

## 문제 해결

### Q: GIF 파일이 너무 큽니다
A: 해상도를 낮추거나 프레임 레이트를 줄이세요
```bash
# 해상도 800px로 낮추기
ffmpeg -i input.mov -vf "fps=10,scale=800:-1" output.gif

# 색상 줄이기
gifsicle -O3 --colors 128 input.gif -o output.gif
```

### Q: GIF 화질이 나쁩니다
A: gifski를 사용하거나 프레임 레이트를 높이세요
```bash
gifski -o output.gif frame*.png --fps 20 --quality 100
```
