## 주요 기능

### 1. 자동 팝업 챗봇
- 페이지 로드 1초 후 자동 표시
- 우측 하단 고정 위치
- 최소화/펼치기 토글

### 2. 고객 매칭
- 고객명 입력 시 자동 매칭
- 퍼지 매칭 지원 (하이닉스 → SK하이닉스)
- 고객 정보 자동 로드

### 3. AI 사고 과정 시각화
- 단계별 메시지 순차 표시
- 복합 질문 분해 표시
- 논리 검증 표시
- 신뢰도 평가 표시

### 4. 실시간 정보 패널
- 고객 정보 (회사명, 제품, 버전, 라이선스)
- 과거 이력 (최근 3건)
- AI 분석 인사이트
- 신뢰도 배지
- 참조 문서

### 5. 담당자 입장 표시
- 4명 담당자 순차 입장
- 프로필 카드 애니메이션
- 진행 상황 표시
- 시스템 메시지

### 6. 반응형 디자인
- 모바일 (< 768px)
- 태블릿 (768px - 1024px)
- 데스크톱 (> 1024px)
- XL 데스크톱 (> 1280px, 정보 패널 표시)

## 데모 시나리오

### 시나리오 1: SK하이닉스 (2분)
**목표**: AI 자동 해결 능력 시연

1. "SK하이닉스" 입력 → 고객 정보 매칭
2. 질문 입력 → AI 사고 과정 표시
3. 답변 생성 → 신뢰도 + 참조 문서
4. 정보 패널에서 과거 이력 확인
5. 10개 질문 빠른 응답

### 시나리오 2: 국방부 (3분)
**목표**: 복합 질문 처리 + 에스컬레이션 시연

1. "국방부" 입력 → 고객 정보 매칭
2. 복합 질문 입력 → 질문 분해 (3개)
3. AI 사고 과정 표시 (9단계)
4. 답변 생성 → "담당자 연결하기" 버튼
5. 버튼 클릭 → 4명 담당자 순차 입장
6. 진행 상황 표시 (1/4 → 4/4)
7. 최종 메시지

## 이슈 및 해결

### 이슈 1: 정보 패널 위치
**문제**: 정보 패널이 챗봇과 겹침
**해결**: XL 화면(> 1280px)에서만 표시, 위치 조정

### 이슈 2: 애니메이션 성능
**문제**: 많은 메시지 시 스크롤 지연
**해결**: CSS 애니메이션 최적화, transform 사용

### 이슈 3: 모바일 반응형
**문제**: 모바일에서 챗봇 크기 과다
**해결**: 반응형 크기 조정, 정보 패널 숨김
