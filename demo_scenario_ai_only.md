# ANY 브릿지 AI 챗봇 데모 시나리오 (AI 전용 모드)

## 개요
- **목적**: AI 챗봇의 지식 기반 답변 능력과 세션 종료 프로세스 시연
- **대상**: SK하이닉스 보안 담당자
- **특이사항**: 에스컬레이션 없이 AI 답변만으로 상담 완료, 3배속 진행

## 시나리오 단계

### 1. 초기 접속 및 정보 입력
- **URL**: https://hackerton-kappa.vercel.app
- **액션**: 챗봇 아이콘 클릭 → 인테이크 폼 입력
  - 기업명: `SK하이닉스`
  - 담당자명: `홍길동`
  - 이메일: `hong@skhynix.com`
  - 버튼: `상담 시작하기` 클릭

### 2. 첫 번째 질문: 제품 개요 및 버전 확인
- **사용자 입력**: `안녕하세요, 마크애니 DRM 솔루션에 대해 궁금합니다. 현재 최신 버전이 무엇인가요?`
- **기대 결과**: AI가 반갑게 인사하며 Document SAFER v3.2가 최신 버전임을 안내 (Thinking Process 노출)

### 3. 두 번째 질문: 기술적 상세 문의 (v3.2 개선점)
- **사용자 입력**: `v3.2에서 성능이나 호환성 측면에서 구체적으로 어떤 점이 개선되었나요?`
- **기대 결과**: 윈도우 11 완벽 지원, 대량 파일 처리 성능 30% 향상 등 구체적 수치와 기술 사양 답변

### 4. 세 번째 질문: 도면 보안 (반도체 도면)
- **사용자 입력**: `반도체 도면 암호화도 필요합니다. 마크애니 DRM이 지원하는 도면 종류(CAD 등)가 구체적으로 어떻게 되나요?`
- **기대 결과**: AutoCAD, SolidWorks, CATIA, NX 등 주요 2D/3D CAD 소프트웨어 지원 및 도면 내 객체 단위 제어 기능 안내

### 5. 상담 종료 및 만족도 확인
- **사용자 입력**: `상세한 답변 감사합니다. 오늘은 여기까지 상담하겠습니다.`
- **기대 결과**: AI가 상담을 마무리하며 추가 문의 사항 확인 및 종료 안내

### 6. 세션 종료 프로세스
- **액션**: (시나리오상 자동 종료 또는 종료 버튼 클릭 시뮬레이션)
- **결과**: "상담 내용을 이메일로 정리하여 발송해 드리겠습니다" 메시지와 함께 세션 종료
